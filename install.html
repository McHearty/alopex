Installation

Build from source:

git clone http://github.com/TrilbyWhite/alopex.git
cd alopex

Edit config.h to customize colors and keybindings and/or select one of the included themes with the MOLT environment variable

MOLT=WinterCoat make
sudo make install

Archlinux (AUR):
alopex-git
Other distros
I'll gladly host packages for other distros. Submit them to the email in the footer.
